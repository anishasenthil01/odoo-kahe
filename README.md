# Form Tracker System

A complete web application for tracking form submissions with WhatsApp notifications.

## Features

- User registration and login with JWT authentication
- Add and manage form links with deadlines
- Track form submissions
- Automatic WhatsApp reminders for upcoming deadlines
- Dashboard with statistics
- PostgreSQL database backend

## Prerequisites

- Node.js (v14 or higher)
- PostgreSQL (v12 or higher)
- npm or yarn package manager
